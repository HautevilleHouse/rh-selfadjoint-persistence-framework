   with `c_*>0`, `e_*>=0`, and strict margin `e_*<c_*A_{ker,*}`.

Appendix A uses fixed theorem inequalities.

**Fixed theorem constants used in this manuscript.**

- exact projected-family comparison constants: `c_* = 1`, `e_* = 0`;
- tube radius: `r_* = 3.3437291061e-4`;

### A.2 Lemma A1 (shape-space reduction)
`range(L_N(y))=1_N^perp`.

**Proof.**  
`L_N` is symmetric positive semidefinite because `L_N=S_N^T W_N S_N` and `W_N` is diagonal positive.  
For symmetric operators, `range(L_N)=ker(L_N)^perp`.  
By assumption `ker(L_N)=span{1_N}`; therefore `range(L_N)=1_N^perp`. `QED.`

### A.3 Lemma A2 (coercive lower extraction)
For every `xi in 1_N^perp`,

`<xi,L_N(y)xi> >= (c_*A_{ker,*}-e_*) ||xi||^2`.

**Proof.**  
Apply comparison inequality:

`<xi,L_N xi> >= c_*<xi,K_{ker,N}xi> - e_*||xi||^2`.

Use kernel floor:

`<xi,K_{ker,N}xi> >= A_{ker,*}||xi||^2`.

Combine:

`<xi,L_N xi> >= (c_*A_{ker,*}-e_*)||xi||^2`. `QED.`

### A.4 Theorem A3 (EG1 projected coercivity)
Define

`kappa_{L,*}:=c_*A_{ker,*}-e_*`.

Then `kappa_{L,*}>0` and, for all admissible `N`, canonical-tube states `y`, and all `xi in 1_N^perp`,

`<xi,E_N(y)xi> >= kappa_{L,*} ||xi||^2`.

**Proof.**  
Positivity follows from `e_*<c_*A_{ker,*}`.  
By Lemma A1 the projected operator is exactly the restriction of `L_N` on `1_N^perp`.  
Apply Lemma A2 on `1_N^perp`. `QED.`

### A.5 Corollary A4 (uniform spectral floor)
`lambda_min(E_N(y)) >= kappa_{L,*}` uniformly on admissible class.

**Proof.**  
Immediate from Theorem A3 by Rayleigh quotient characterization. `QED.`

### A.5A Lemma A5 (exact projected-family comparison constants)
Define `Pi_perp,N := I - (1/d_N) 1_N 1_N^T` and

`K_(proj,N)(u) := Pi_perp,N L_N(u) Pi_perp,N`.

Then for all `xi in 1_N^perp`:

`<xi,L_N(u)xi> = <xi,K_(proj,N)(u)xi>`.

Hence comparison constants can be taken exactly as:

`c_* = 1`, `e_* = 0`.

**Proof.**  
If `xi in 1_N^perp`, then `Pi_perp,N xi = xi`. Therefore

`<xi,K_(proj,N)xi> = <xi,Pi_perp,N L_N Pi_perp,N xi> = <xi,L_N xi>`.

So lower/upper comparison inequalities are equalities. `QED.`

### A.5B Theorem A6 (full-chart frozen-floor + Lipschitz closure)
Let `K_(proj,N)^(full)(u) := Pi_perp,N L_N(u) Pi_perp,N`.
Suppose there exist constants `mu_(ker,0)^(full)>0`, `L_(ker,*)^(full)<infinity`, `r_*>0`, independent of `N`, such that:

1. `lambda_min( K_(proj,N)^(full)(u_(N,*))|_(1_N^perp) ) >= (mu_(ker,0)^(full))^2`,
2. `||D K_(proj,N)^(full)(u)|| <= L_(ker,*)^(full)` on canonical tube,
3. `L_(ker,*)^(full) r_* < (mu_(ker,0)^(full))^2`.

Then

`A_*^(full) := (mu_(ker,0)^(full))^2 - L_(ker,*)^(full) r_* > 0`,

and for every admissible `N`, tube state `u`, and `xi in 1_N^perp`:

`<xi,L_N(u)xi> >= <xi,K_(proj,N)^(full)(u)xi> >= A_*^(full)||xi||^2`.

**Proof.**  
By Weyl perturbation:

`lambda_min(K_(proj,N)^(full)(u)|_(1_N^perp))`
`>= lambda_min(K_(proj,N)^(full)(u_(N,*))|_(1_N^perp)) - L_(ker,*)^(full) r_*`
`>= (mu_(ker,0)^(full))^2 - L_(ker,*)^(full) r_* = A_*^(full)`.

Positivity follows from item (3).  
On `1_N^perp`, projector idempotence gives equality between projected and restricted quadratic forms. `QED.`

### A.5C Proposition A7 (response-map instantiation of full constants)
Let `mathcal L_full,N` be the centered shifted-log response map and

`M_(ell,N)^(full)(u):=D mathcal L_full,N(u)`,

`G_(ell,resp,N)^(full)(u):=M_(ell,N)^(full)(u)(M_(ell,N)^(full)(u))^*`.

1. `sigma_min^+(M_(ell,N)^(full)(u_(N,*))) >= mu_(ell,0)^(full) > 0`,
2. `||M_(ell,N)^(full)|| <= L_(ell,full,*)`,
3. `||D M_(ell,N)^(full)|| <= H_(ell,full,*)`,
4. `K_(proj,N)^(full) = G_(ell,resp,N)^(full)` on shape space.

Then one may take

`mu_(ker,0)^(full):=mu_(ell,0)^(full)`,

`L_(ker,*)^(full):=2 L_(ell,full,*) H_(ell,full,*)`,

so

`A_*^(full)=(mu_(ell,0)^(full))^2 - 2 L_(ell,full,*)H_(ell,full,*) r_*`.

**Proof.**

`lambda_min(MM^*) = sigma_min(M)^2`.

Derivative bound:

`D(MM^*)[h]=DM[h]M^* + M(DM[h])^*`,
so
`||D(MM^*)[h]|| <= 2||M|| ||DM[h]|| <= 2 L_(ell,full,*) H_(ell,full,*) ||h||`.

Use identification `K_(proj,N)^(full)=MM^*`. `QED.`

### A.5E Theorem A9 (non-proxy inequality form of full-chart closure)

### B.1 Setup

- strict coherence lane: `eta_{coh,*}=0`.

Appendix B is stated from explicit theorem constants/inequalities.

### B.2 Lemma B1 (flow-segment invariant quantity)
On any open flow interval `(a,b)` with no restart,
the quantity

`Phi(t):=exp(L_D t)*(D(t)-sigma_*+C(t))`

is nondecreasing.

**Proof.**  
Differentiate on `(a,b)`:

`d/dt Phi = exp(L_D t)[d/dt(D-sigma_*+C) + L_D(D-sigma_*+C)]`.

Using the assumed inequality:

`d/dt(D-sigma_*+C) >= -L_D(D-sigma_*)`.

Hence

`d/dt Phi >= exp(L_D t)*L_D*C(t) >= 0`,

because `C(t) >= 0`. `QED.`

### B.3 Lemma B2 (segment lower bound)
On `(a,b)`:

`D(t) >= sigma_* - C(t) + exp(-L_D(t-a))*(D(a)-sigma_*+C(a))`.

In particular, if `D(a) >= sigma_* - C(a)`, then `D(t) >= sigma_* - C(t)`.

**Proof.**  
From Lemma B1, `Phi(t) >= Phi(a)`; divide by `exp(L_D t)`. `QED.`

### B.4 Lemma B3 (restart preservation)
At each restart `tau_k`,

`D(tau_k^+) >= sigma_* >= sigma_* - C(tau_k^+)`.

**Proof.**  
Direct from restart capture and `C>=0`. `QED.`

### B.5 Theorem B4 (EG2 capture-invariance)

`D(t) >= sigma_* - C(t) = sigma_* - Delta_coh[t0,t]`.

If `Delta_coh[t0,t]=0` (strict coherence), then `D(t) >= sigma_*` for all times.

**Proof.**  
Start at `t0`: `D(t0) >= sigma_* = sigma_* - C(t0)`.  
Propagate through first flow interval using Lemma B2.  
At restart use Lemma B3, then repeat by induction over intervals/restarts. `QED.`

### E.2 Theorem E1 (determining-class arithmetic lock)
Let `mu_0`, `mu_Xi` be Borel probability measures on `[0,1]`.  
Suppose

`int f dmu_0 = int f dmu_Xi` for all `f in mathcal F`,

where `mathcal F` is determining on `[0,1]` (equivalently, linear span uniformly dense in `C([0,1])`).

Then `mu_0 = mu_Xi`.

In particular this holds if `mathcal F` is:

1. polynomial class `{x^k}_{k>=0}`, or
2. Cauchy-kernel class `{(x-z)^(-1): z in S}`, `S subset C\\[0,1]` with an accumulation point.

**Proof.**  
Equality on `mathcal F` extends by linearity and uniform approximation to all continuous test functions on `[0,1]`.  
By Riesz representation, equality of integrals on all `C([0,1])` implies equality of measures.  
For class (1), use Weierstrass density.  
For class (2), equality of Cauchy transforms on a set with accumulation point forces equality by analytic continuation, hence equal Stieltjes transforms and measures. `QED.`
